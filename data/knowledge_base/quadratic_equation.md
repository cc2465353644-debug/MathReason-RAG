# 一元二次方程

## 1. 知识点说明

一元二次方程是指只含有一个未知数，并且未知数最高次数为 2 的方程。一般形式为：
```text
ax^2 + bx + c = 0, a ≠ 0
```
常见解法包括因式分解法、配方法和求根公式法。对于基础题，优先尝试因式分解；如果不能直接分解，可以使用求根公式。

### 2. 常用公式

判别式：
```text
Δ = b^2 - 4ac
```
求根公式：
```text
x = (-b ± sqrt(b^2 - 4ac)) / 2a
```
根的情况：
```text
Δ > 0：两个不相等实根
Δ = 0：两个相等实根
Δ < 0：无实数根
```
### 3. 适用题型

- 解一元二次方程
- 判断方程根的个数
- 根据方程根求参数
- 因式分解相关题目

### 4. 解题步骤

1. 将方程整理为标准形式；
2. 判断是否可以因式分解；
3. 如果可以分解，令每个因式等于 0；
4. 如果不能分解，使用求根公式；
5. 写出最终答案。

### 5. 例题

题目：
```text
解方程：x^2 - 5x + 6 = 0
```
解答：
```text
x^2 - 5x + 6 = (x - 2)(x - 3)
所以：
x - 2 = 0 或 x - 3 = 0
因此：
x = 2 或 x = 3
```
最终答案：
```text
x = 2 或 x = 3
```
