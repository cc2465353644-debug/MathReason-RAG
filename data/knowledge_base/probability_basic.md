# 概率基础

## 1. 知识点说明

概率用于描述随机事件发生的可能性。在基础题中，常见的是古典概型，即所有基本结果出现的可能性相同。
古典概型的概率计算公式为：
```text
P(A) = 事件 A 包含的基本结果数 / 所有可能基本结果数
```
概率的取值范围为：
```text
0 ≤ P(A) ≤ 1
```
如果事件一定发生，则概率为 1；如果事件不可能发生，则概率为 0。

## 2. 常用公式
古典概型：
```text
P(A) = m / n
```
其中：
```text
m 表示事件 A 有利结果数
n 表示所有可能结果数
```
对立事件：
```text
P(A') = 1 - P(A)
```
两个互斥事件：
```text
P(A 或 B) = P(A) + P(B)
```

## 3. 适用题型

-  抽球问题
- 掷骰子问题
- 抛硬币问题
- 简单古典概型
- 求对立事件概率

## 4. 解题步骤

1. 找出所有可能结果数；
2. 找出符合条件的结果数；
3. 使用概率公式计算；
4. 根据需要将结果写成分数、小数或百分数；
5. 检查概率是否在 0 到 1 之间。

## 5. 例题

题目：
```text
一个袋子中有 3 个红球和 2 个蓝球，随机取出 1 个球，取到红球的概率是多少？
```
解答：
```text
袋子中共有：
3 + 2 = 5 个球
其中红球有 3 个。
所以取到红球的概率为：
P = 3 / 5
```
最终答案：
```text
3/5
```
也可以写成：
```text
0.6
```
