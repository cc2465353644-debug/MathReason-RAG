# 极限基础

## 1. 知识点说明

极限用于描述变量趋近某个值时，函数值的变化趋势。基础极限题通常可以通过直接代入、因式分解、约分、通分或使用常见极限公式解决。
如果直接代入后分母不为 0，一般可以直接计算函数值。如果出现 0/0 形式，通常需要先化简。

## 2. 常用方法

直接代入：
```text
如果 f(x) 在 x = a 处连续，则 lim(x→a) f(x) = f(a)
```
因式分解约分：
```text
当出现 0/0 时，可以尝试因式分解并约去公共因式
```
常见重要极限：
```text
lim(x→0) sin x / x = 1
lim(x→0) (1 - cos x) / x = 0
lim(x→0) (e^x - 1) / x = 1
```
## 3. 适用题型

- 求函数在某点处的极限
- 多项式或分式函数极限
- 简单三角函数极限
- 判断函数连续性

## 4. 解题步骤

1. 先尝试直接代入；
2. 如果能得到确定值，直接写答案；
3. 如果出现 0/0，需要化简；
4. 常见方法包括因式分解、约分、通分；
5. 化简后再次代入求值。

## 5. 例题
题目：
```text
计算极限：lim(x→2) (x^2 - 4) / (x - 2)
```
解答：
```text
直接代入 x = 2，会得到 0/0，不能直接计算。
先因式分解：
x^2 - 4 = (x - 2)(x + 2)
所以：
(x^2 - 4) / (x - 2)
= (x - 2)(x + 2) / (x - 2)
= x + 2
因此：
lim(x→2) (x^2 - 4) / (x - 2)
= lim(x→2) (x + 2)
= 4
```
最终答案：
```text
4
```
