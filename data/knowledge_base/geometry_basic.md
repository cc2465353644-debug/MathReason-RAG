# 几何基础

## 1. 知识点说明

几何基础题通常涉及常见图形的周长、面积、角度和边长关系。常见图形包括三角形、矩形、正方形、圆等。
在基础数学题中，最常见的是根据已知条件计算面积、周长或利用勾股定理求边长。

## 2. 常用公式

矩形面积：
```text
S = 长 × 宽
```
正方形面积：
```text
S = 边长^2
```
三角形面积：
```text
S = 底 × 高 / 2
```
圆的周长：
```text
C = 2πr
```
圆的面积：
```text
S = πr^2
```
勾股定理：
```text
a^2 + b^2 = c^2
```
其中 c 是直角三角形的斜边。

## 3. 适用题型

- 求图形面积
- 求图形周长
- 求圆的面积或周长
- 直角三角形边长计算
- 简单几何综合题

## 4. 解题步骤

1. 判断题目涉及的图形；
2. 找出已知量；
3. 选择对应公式；
4. 代入数值计算；
5. 写出结果并注意单位。

## 5. 例题

题目：
```text
半径为 3 的圆的面积是多少？
```
解答：
```text
圆的面积公式为：
S = πr^2
代入 r = 3：
S = π * 3^2
= 9π
```
最终答案：
```text
9π
```

## 6. 注意事项

几何题中要注意单位。如果题目中的长度单位是厘米，则面积单位应为平方厘米；如果长度单位是米，则面积单位应为平方米。
