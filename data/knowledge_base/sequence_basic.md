# 数列基础

## 1. 知识点说明

数列是按照一定顺序排列的一列数。基础数学题中最常见的是等差数列和等比数列。
等差数列的相邻两项差相等；等比数列的相邻两项比值相等。

## 2. 等差数列

如果数列每一项与前一项的差都相同，则称为等差数列。这个固定差值称为公差，记作 d。
通项公式：
```text
a_n = a_1 + (n - 1)d
```
前 n 项和公式：
```text
S_n = n(a_1 + a_n) / 2
```
也可以写成：
```text
S_n = n[2a_1 + (n - 1)d] / 2
```

## 3. 等比数列

如果数列每一项与前一项的比值都相同，则称为等比数列。这个固定比值称为公比，记作 q。
通项公式：
```text
a_n = a_1 q^(n-1)
```
前 n 项和公式：
```text
S_n = a_1(1 - q^n) / (1 - q), q ≠ 1
```

## 4. 适用题型

- 求数列通项
- 求前 n 项和
- 判断等差或等比数列
- 根据已知项求公差或公比

## 5. 例题

题目：
```text
已知等差数列 a_1 = 3，公差 d = 2，求 a_5。
```
解答：
```text
等差数列通项公式为：
a_n = a_1 + (n - 1)d
代入 a_1 = 3，d = 2，n = 5：
a_5 = 3 + (5 - 1) * 2
= 3 + 8
= 11
```
最终答案：
```text
a_5 = 11
```
