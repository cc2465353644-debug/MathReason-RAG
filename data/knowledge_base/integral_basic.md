# 不定积分基础

## 1. 知识点说明

积分可以看作求导的逆运算。在基础题中，常见任务是根据函数求它的原函数。不定积分的结果通常要加上积分常数 C。
对于幂函数，可以使用幂函数积分公式。

## 2. 常用公式

常数积分：
```text
∫ k dx = kx + C
```
幂函数积分：
```text
∫ x^n dx = x^(n+1)/(n+1) + C, n ≠ -1
```
常数倍法则：
```text
∫ kf(x) dx = k∫f(x) dx
```
和差法则：
```text
∫ [f(x) ± g(x)] dx = ∫f(x) dx ± ∫g(x) dx
```
常见积分：
```text
∫ 1/x dx = ln|x| + C
∫ e^x dx = e^x + C
∫ cos x dx = sin x + C
∫ sin x dx = -cos x + C
```
## 3. 适用题型

- 求简单不定积分
- 求多项式函数的原函数
- 验证积分结果是否正确-
-  与导数结合的基础题

## 4. 解题步骤

1. 识别被积函数；
2. 将复杂表达式拆成若干简单项；
3. 对每一项分别积分；
4. 合并结果；
5. 不要忘记加积分常数 C。

## 5. 例题

题目：
```text
计算不定积分：∫ 2x dx
```
解答：
```text
根据幂函数积分公式：
∫ x dx = x^2 / 2
所以：
∫ 2x dx = 2 * x^2 / 2 = x^2
不定积分需要加上常数 C。
```
最终答案：
```text
x^2 + C
```
