# 导数基础

## 1. 知识点说明

导数用于描述函数在某一点附近的变化率，也可以理解为函数图像切线的斜率。在基础数学题中，常见任务包括求函数导数、判断函数单调性、求极值等。
如果函数为多项式，可以使用幂函数求导公式逐项求导。

## 2. 常用公式

常数求导：
```text
(C)' = 0
```
幂函数求导：
```text
(x^n)' = n x^(n-1)
```
常数倍法则：
```text
(kf(x))' = kf'(x)
```
和差法则：
```text
(f(x) ± g(x))' = f'(x) ± g'(x)
```
常见函数导数：
```text
(sin x)' = cos x
(cos x)' = -sin x
(e^x)' = e^x
(ln x)' = 1/x
```
## 3. 适用题型

- 求函数导数
- 求切线斜率
- 判断函数单调性
- 求函数极值
- 分析函数变化趋势

## 4. 解题步骤

1. 判断函数类型；
2. 对每一项分别求导；
3. 使用求导法则合并结果；
4. 如果题目要求单调性，需要分析导数正负；
5. 写出最终答案。

## 5. 例题

题目：
```text
求函数 f(x) = x^3 - 2x + 1 的导数。
```
解答：
```text
f(x) = x^3 - 2x + 1
根据幂函数求导公式：
(x^3)' = 3x^2
(-2x)' = -2
(1)' = 0
所以：
f'(x) = 3x^2 - 2
```
最终答案：
```text
f'(x) = 3x^2 - 2
```
